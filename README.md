# Modern Expense Tracker

A clean, modern, single-file expense tracking application designed to help you monitor your spending habits with ease. Built with simplicity and aesthetics in mind.

## Features

-   **Easy Entry**: Quickly add expenses with description, amount, and category.
-   **Visual Analytics**: Real-time spending breakdown using an interactive doughnut chart (powered by Chart.js).
-   **Categorization**: Organize expenses into Food, Transport, Entertainment, and Other.
-   **Data Persistence**: Your data is saved automatically to your browser's LocalStorage, so it won't be lost when you refresh or close the page.
-   **Responsive Design**: Works beautifully on desktop and mobile devices.

## How to Run

Since this is a single-file application, running it is incredibly simple:

1.  Clone the repository:
    ```bash
    git clone https://github.com/chlodomer/expense-tracker.git
    ```
2.  Navigate to the project folder.
3.  Open `index.html` in your favorite web browser.

Alternatively, you can preview it locally using a simple Python server:
```bash
python3 -m http.server
```
